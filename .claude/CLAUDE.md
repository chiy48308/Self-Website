# Project Instructions

## Skills

### frontend-design
Trigger this skill when the user asks to build web components, pages, artifacts, posters, or applications (e.g. websites, landing pages, dashboards, React components, HTML/CSS layouts, or when styling/beautifying any web UI).

Full skill instructions: [.claude/frontend-design/SKILL.md](.claude/frontend-design/SKILL.md)

**Key principles:**
- Commit to a bold, intentional aesthetic direction before coding
- Avoid generic AI aesthetics (no Inter/Roboto/Arial, no purple gradients on white, no cookie-cutter layouts)
- Use distinctive typography, cohesive color palettes, purposeful motion, and creative spatial composition
- Produce production-grade, functional, visually memorable code
- Every design should feel unique and context-specific
